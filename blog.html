Question-1: What are the different ways to select an element in the DOM?

We can select an element in the DOM in different ways :
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

We use innerHTML if we need to update or read HTML content. But we use the innerText when we only use the visible text like in a p tag or h1 to h6 tags and we use the textContent when we want all text,all the elements and don't need HTML formatting.

Question-3: What is event delegation in the DOM?

Event delegation is a way to handle events more efficiently.Instead of adding event listeners to multiple elements, we add one listener to a parent element and check which child was clicked.

Question-4: What is event bubbling in the DOM?

Event bubbling means when we click on an element, the event first happens on that element and then goes to its parent elements. WE can stop this using event.stopPropagation().

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element using document.createElement("tag").
To add it to the page, use appendChild().
To remove an element, use element.remove() or parent.removeChild(child).
